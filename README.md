## For JetBrains Team

This project is a migration of the Kotlin site frontend to React Router (Vite + React 19). The app runs in development and production with server-side rendering. Below are the steps to install dependencies and run the project so you can review the work locally.

---

## Prerequisites

- **Node.js** — version **18** or **20** 
  Check: `node -v`
- **npm** — comes with Node.js.  
  Check: `npm -v`

---

## Installation

1. **Clone the repository** (if you don’t have it yet):
   ```bash
   git clone <repository-url>
   cd kotlin-intern
   ```

2. **Install dependencies**:
   ```bash
   npm install/ yarn/ pnpm install
   ```
3. 
4. **Run the app**:
   ```bash
   npm run dev
   ```
   The dev server will start (e.g. `http://localhost:5173`). Open this URL in a browser to view the site.
